## 深度监听
* 需要一次监听到底，计算量大
* 无法监听新增、删除属性,
* 无法监听数组

# 虚拟DOM
* VUE的虚拟DOM 基于 https://github.com/snabbdom/snabbdom
```vue
//html
<div id="div1" class="class1">
    <p>vdom</p>
    <ul style="font-size: 20px">
        <li>a</li>
    </ul>
</div>

//vdom
{
    tag: 'div',
     props: {
        className:'class1',
        id: 'id1',
     },
    children: [
        {
            tag: 'p',
            clildren: 'vdom'
        },
        {
            tag: 'ul',
            props: {
                style: "font-size: 20px"
            },
            clildren: [
                {
                    tag: 'li',
                    children: 'a'
                }
            ]
        }
    ]
}
```
